Welcome to DARC’s documentation¶

DARC (Data Analysis of Real-time Candidates) is the real-time processing pipeline of the Apertif Radio Transient System (ARTS). DARC consists of several services, that each execute a subset of the pipeline. The pipeline consists of four global parts:

  1. Real-time triggering based on metadata of candidate transients

  2. Real-time data processing of candidate transients

  3. Offline post-processing

  4. Monitoring & control

For more information about these different parts of DARC, consult one of the following pages:

  • Real-time triggering
  • Real-time data processing
  • Offline data processing
  • Monitoring & control
  • API reference
